# Information Security Policy Framework

## 1. Purpose
This document defines the Information Security Policy Framework for the organisation.
It establishes a structured hierarchy of security documentation to ensure confidentiality,
integrity, and availability of information assets.

## 2. Scope
This framework applies to:
- All employees, contractors, and third parties
- All information systems and data assets
- All physical and digital environments

## 3. Policy Hierarchy

### Level 1 – Information Security Policy
Defines executive intent, governance principles, and organisational commitment to security.

### Level 2 – Security Standards
Define mandatory security requirements aligned to:
- ISO/IEC 27001
- NIST Cybersecurity Framework
- GDPR

### Level 3 – Procedures
Step-by-step instructions for implementing security controls.

### Level 4 – Guidelines
Recommended practices to support secure behaviour.

### Level 5 – Records
Evidence demonstrating compliance (logs, reports, registers).

## 4. Review & Maintenance
This framework is reviewed annually or following significant organisational change.

## 5. Approval
Approved by: Board of Directors  
Owner: Chief Information Security Officer (CISO)
